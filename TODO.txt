Topic List Page
---------------
	Open last page[option]: /Main#CommMsgs?cmm={community.id}&tid={topic.id}&na=2&nst={topic.lastReadMsg - 9}


TopicMessagesPage
-----------------
	User options:
		Go to next page when marking current page as read (if current page is not the last one).

	Display(?) something like:
		"X unread messages in the next pages"
		"X unread messages in the previous pages"


General
-------
	Store data in incognito mode (default: no).

	Add Orkut username to the key:
		localStorage[username + ':' + topic.id]
